== Description ==

Adds Masked input to ACF.

= Compatibility =

This ACF field type is compatible with:
* ACF 5

== Installation ==

1. Copy the `acf-maskfield` folder into your `wp-content/plugins` folder
2. Activate the Mask Field plugin via the plugins admin page
3. Create a new field via ACF and select the Mask Field type
4. Read the description above for usage instructions
